Mariner's Atlas — Guide

Quick reference for navigating the atlas
Adding Ports
• Click anywhere on the map to anchor a new port
• Enter a name in the dialog (or leave blank for auto-numbering)
• Ports display with permanent labels showing their names
• Click any port to edit its details, add notes, or delete it
Drawing Sea Routes
• Wait for "Sea charts loaded" to appear in the control panel
• Click the Draw Route button
• Click two ports to chart a maritime route between them
• The app uses A* pathfinding to plot realistic sea routes that avoid land masses and navigate through straits (Panama, Suez, Gibraltar, etc.)
• If two significantly different routes exist, you'll be prompted to choose your preferred course
• Routes show distance in km and nautical miles when clicked
Port Info Bubbles
• Click any port to open its details
• Add notes in the "Notes & Cargo" field (cargo manifest, hazards, weather, etc.)
• Check "Display info bubble on map" to show a callout
• The bubble appears as a draggable speech balloon connected to the port
• Drag the bubble to reposition it anywhere on the map
• Uncheck the box and save to hide the bubble
Navigator Tools
• Click Find Port to search by port name or saved notes
• Selecting a match jumps the map to that port and briefly highlights it
• Click Measure Distance to start the straight-line ruler
• Click any two points on the map to measure great-circle distance in km and nautical miles
• Measurements are visual aids only and are not included in JSON exports
Exporting as PNG
• Click Export PNG in the Logbook section
• The map is captured without UI controls, showing only ports, labels, routes, and info bubbles
• The PNG is saved with a timestamp: mariners-atlas-YYYY-MM-DD...
• High-resolution output suitable for printing or sharing
Exporting as JSON
• Click Export JSON to save your entire atlas
• The JSON file includes all ports (names, coordinates, notes, bubble positions), all routes (exact paths, distances), and the current map view
• Save this file to back up your work or share it with others
• Filename format: mariners-atlas-YYYY-MM-DD...
Importing from JSON
• Click Import JSON and select a previously exported atlas file
• The current map will be cleared and replaced with the imported data
• All ports, routes, notes, and bubble positions are restored exactly as saved
• The camera view (zoom and center) is also restored
• Use this to continue working on a saved atlas or load someone else's routes
Tips
• Routes use real maritime pathfinding — expect ships to navigate around continents and through major canals
• Press Esc to close any open modal
• Press Esc while drawing or measuring to cancel the active tool
• Deleting a port also removes any routes connected to it
• Info bubbles stay anchored to their ports when dragged — perfect for annotation workflows
• Light/dark mode follows your system preference automatically

Clear the Atlas?
