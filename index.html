Open Atlas — Guide

Quick reference for navigating the atlas
Adding Points
• Click anywhere on the map to anchor a new port or location
• Enter a name in the dialog (or leave blank for auto-numbering)
• Choose a point icon and color while creating the point, or refine them later in details
• Saved points display with permanent labels showing their names
• Click any point to edit its details, add notes, or delete it
Routes and Connections
• In Maritime mode, wait for "Sea charts ready" to appear, then click Chart route
• Click two ports to chart a maritime route between them
• The app uses A* pathfinding to plot realistic sea routes that avoid land masses and navigate through straits (Panama, Suez, Gibraltar, etc.)
• If two significantly different maritime routes exist, you'll be prompted to choose your preferred course
• In Connections mode, click Add connection and choose any two saved locations
• Use Appearance → Connection Style to switch between clean straight links and curved presentation arcs
• Routes and connections both show distance in km and nautical miles when clicked
Point Info Bubbles
• Click any point to open its details
• Add notes in the "Notes & Cargo" field (cargo manifest, hazards, weather, etc.)
• Check "Display info bubble on map" to show a callout
• The bubble appears as a callout anchored to the point so it reads like an annotation, not a floating card
• Drag the bubble to reposition it relative to the point and keep important map details visible
• Use the small handle in the lower-right corner to resize the bubble in both width and height
• Uncheck the box and save to hide the bubble
Navigator Tools
• Use Find a saved location in the Explore section to search by name or notes in this atlas
• Selecting a match jumps the map to that point and briefly highlights it
• Click Measure distance to start the straight-line ruler
• Click any two points on the map to measure great-circle distance in km and nautical miles
• Measurements are visual aids only and are not included in JSON exports
Exporting as PNG
• Click Export PNG in the Save & Share section
• PNG exports now capture only the map view, without the studio sidebar
• Capture either the current viewport or automatically fit all saved points, labels, bubbles, and routes into frame
• Pick standard, high, or ultra output quality depending on how heavy you want the file to be
• The PNG is saved with a timestamp: open-atlas-YYYY-MM-DD...
• High-resolution output suitable for printing or sharing
Copying the View
• Click Copy View to place the current map image directly on your clipboard
• This uses the current viewport and excludes the studio sidebar by default
• Use it for quick pasting into chat, notes, slides, or documents without downloading a file first
Exporting as JSON
• Click Export JSON to save your entire atlas
• The JSON file includes all saved points (names, coordinates, notes, icons, colors, bubble positions), all routes or connections (exact paths, distances), and the current map view
• Save this file to back up your work or share it with others
• Filename format: open-atlas-YYYY-MM-DD...
Exporting as GeoJSON
• Click Export GeoJSON to create an interoperable map data file
• Saved points become point features and routes or connections become line features
• GeoJSON is ideal for GIS tools, notebooks, or downstream custom workflows
• The file also carries atlas metadata and theme settings
Importing from JSON
• Click Import atlas and select a previously exported atlas file
• The current map will be cleared and replaced with the imported data
• All saved points, routes or connections, notes, and bubble positions are restored exactly as saved
• The camera view (zoom and center) is also restored
• Use this to continue working on a saved atlas or load someone else's map
Studio Settings
• Open Appearance in the Workspace section to tune fonts, colors, theme mode, and the basemap style
• Use Map Mode to switch between maritime routing, location-only pins, and straight-line connections
• Choose Presentation Flat when you want a cleaner editorial export with custom sea and land colors
• The live preview card shows how your title, palette, and typography will read before you export
• Atlas title and subtitle are part of your saved JSON and GeoJSON metadata
• Theme choices are also stored in your browser so the workspace reopens with your preferred look
Undo & Redo
• Use Undo and Redo in the Workspace section to step through atlas edits
• Keyboard shortcuts: Cmd/Ctrl+Z to undo, Shift+Cmd/Ctrl+Z or Ctrl+Y to redo
• Imports, point edits, route changes, and visual settings all participate in history
Tips
• Use the collapse button in the studio header to tuck the panel away, then reopen it from the same edge of the screen
• On phones and tablets, the studio behaves like a bottom sheet so the map stays usable without losing the controls
• The three workflow chips near the top of the sidebar guide you through adding points, linking them when relevant, and exporting assets
• Maritime routes use real pathfinding around continents and through major canals, while Connections mode keeps links straight and lightweight
• Press Esc to close any open modal
• Press Esc while drawing or measuring to cancel the active tool
• Deleting a point also removes any routes or connections attached to it
• Info bubbles stay anchored to their points when dragged or resized, and the typography compacts as the bubble gets smaller
• Light/dark mode follows your system preference automatically
