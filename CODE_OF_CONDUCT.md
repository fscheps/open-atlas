# Code of Conduct

Open Atlas is meant to be a welcoming project.

## Our standard

Please be respectful, constructive, and kind.

We want this project to be a place where people can:

- ask beginner questions safely
- disagree without hostility
- contribute ideas and code in good faith
- collaborate across different experience levels

## Expected behavior

- Be courteous and patient.
- Give feedback about the work, not the person.
- Assume good intent unless there is clear reason not to.
- Help keep discussions practical, inclusive, and focused.
- Respect different backgrounds, identities, and skill levels.

## Unacceptable behavior

- Harassment, abuse, or intimidation
- Personal attacks or insulting language
- Discriminatory or exclusionary behavior
- Deliberate disruption, trolling, or bad-faith participation
- Publishing private information about others without permission

## Scope

This applies to project spaces, including issues, pull requests, discussions, and other community interactions related to Open Atlas.

## Maintainer responsibility

Project maintainers may remove, edit, or reject comments, issues, pull requests, or other contributions that do not align with this code of conduct.

## Reporting

If something needs attention, contact the maintainer privately through the repository owner’s public contact channel or GitHub profile.

## Enforcement

Responses may include warnings, moderation of discussions, removal of content, or banning repeat offenders from project spaces.
